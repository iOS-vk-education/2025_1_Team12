# 2025_1_Team12
Проект команды Team 12 по курсу "Мобильный разработчик на iOS" от VK

## Описание
> GymApp — приложение для сопровождения пользователя во время тренировок. Выбери или создай тренировочный план и начинай заниматься, а приложение соберет все статистику и покажет Ваш прогресс!
